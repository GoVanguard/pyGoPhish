PyGoPhish - A Phishy Web Application

PyGoPhish is a GoVanguard web application that automates phishing reconnaissance and message delivery. PyGoPhish assists the user with targeting a web domain, generating a list of e-mail addresses, developing an e-mail template, and delivering the custom e-mail to the selected domain. PyGoPhish was built using the Django Web Framework, Microsoft Graph, and Python. Many thanks go to Jason Johnston, who is a developer with the Microsoft Graph development team, for helping GoVanguard learn how to leverage Azure's native resources to automate our business logic. This application wouldn't be possible without his team's dedicated work creating the Microsoft Graph Developer Experience and its affiliated training repositories. Johnston, Jason et al.
